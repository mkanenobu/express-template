# Express Template

## Commands

Run development mode

This is the same as parallel execute of `npm run build:dev` and `npm run start:dev`

```bash
npm run dev
```

Production build

```bash
npm run build
```

Run production build

```bash
npm start
```

Run test

```bash
npm run test
```

Run type check

```bash
npm run tc
```

## Tech Stack

- Express
- TypeScript
- Vitest
- esbuild
