# Express Template

## Commands

Run development mode

```bash
npm run dev
# with port number
PORT=5000 npm run dev
```

Production build

```bash
npm run build
# With port number, need to specify port on build time
# https://vitejs.dev/guide/env-and-mode.html#production-replacement
PORT=5000 npm run build
```

Run production build

```bash
npm start
```

Run test

```bash
npm run test
```

Run type check

```bash
npm run tc
```

## Tech Stack

- Express
- TypeScript
- Vitest
- esbuild
